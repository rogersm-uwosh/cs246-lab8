# CS 246 - Lab 8: Data Processing and File I/O

## Overview
This lab focuses on working with different data formats and file input/output operations. You'll practice reading, processing, and analyzing data from various file formats.

## Objectives
- Read and parse different data formats (CSV, JSON, XML)
- Process and analyze data
- Implement file I/O operations
- Handle errors and edge cases

## Lab Structure

### Data Files
- `data/students.csv` - Student records in CSV format
- `data/courses.json` - Course information in JSON format
- `data/grades.xml` - Grade records in XML format
- `data/sample.txt` - Plain text sample data

### Tasks
1. **Task 1**: Read student data from CSV file
2. **Task 2**: Parse course information from JSON
3. **Task 3**: Process grade records from XML
4. **Task 4**: Generate summary reports

## Getting Started
1. Clone this repository
2. Choose your preferred programming language
3. Implement functions to read and process each data file
4. Test your implementation with the provided data

## Expected Outputs
- Student roster sorted by name
- Course enrollment statistics
- Grade distribution analysis
- Combined report in a specified format

## Resources
- File I/O documentation for your chosen language
- CSV/JSON/XML parsing libraries
- Sample code in `examples/` directory

Submit your source code files along with output files to the course portal.
